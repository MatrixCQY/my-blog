# 股票基础知识

## 什么是股票？

股票是股份公司发行的所有权凭证，代表股东对公司的所有权。持有股票意味着你拥有公司的一部分。

## 股票市场

### A股市场

A股是指在中国大陆注册、在中国大陆上市的普通股票，以人民币认购和交易。

### 主要指数

- **上证指数（000001）**：反映上海证券交易所股票价格变动情况
- **深证成指（399001）**：反映深圳证券交易所股票价格变动情况
- **创业板指（399006）**：反映创业板市场股票价格变动情况

## 股票分类

### 按行业分类

- **金融股**：银行、保险、证券
- **科技股**：互联网、软件、硬件
- **消费股**：食品饮料、家电、零售
- **医药股**：制药、医疗器械
- **能源股**：石油、煤炭、电力

### 按市值分类

- **大盘股**：市值超过500亿
- **中盘股**：市值在100-500亿之间
- **小盘股**：市值小于100亿

## 基本术语

- **开盘价**：交易日第一笔成交价格
- **收盘价**：交易日最后一笔成交价格
- **最高价**：交易日最高成交价格
- **最低价**：交易日最低成交价格
- **成交量**：交易日成交的股票数量
- **换手率**：成交量与流通股本的比率

## 交易时间

- **集合竞价**：9:15-9:25
- **连续竞价**：9:30-11:30, 13:00-15:00
- **收盘集合竞价**：14:57-15:00（仅深交所）

## 注意事项

1. **风险提示**：股市有风险，投资需谨慎
2. **分散投资**：不要把所有资金投入单一股票
3. **长期投资**：避免频繁交易，关注长期价值
4. **学习研究**：持续学习，提高投资能力
